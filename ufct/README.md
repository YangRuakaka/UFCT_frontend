# 🎓 学术网络可视化前端项目

**一个完整的、生产级别的学术网络交互式可视化系统**

## 📌 项目概览

本项目是一个全功能的前端应用，用于交互式可视化计算机科学学术网络。包括论文引用网络和作者协作网络两个主要模块，采用 Vue 3 + D3.js 技术栈，具有完善的性能优化和丰富的交互功能。

**项目状态**：✅ 生产级别 - 可立即使用

## 🚀 快速开始（5 分钟）

```bash
# 进入项目目录
cd d:\Github\UFCT_frontend\ufct

# 启动开发服务器
npm run serve

# 打开浏览器访问
http://localhost:8080
```

详见 **QUICKSTART.md** 或 **FRONTEND_README.md**

## 🎯 主要功能

### 论文引用网络 📄
- 显示论文之间的引用关系
- 节点大小代表被引用频率
- 完整的交互和信息展示

### 作者协作网络 👥
- 显示作者之间的协作关系
- 支持与论文网络相同的所有功能
- 动态数据加载

### 交互功能 🖱️
```
悬停节点     → 高亮相关连接
点击节点     → 显示详细信息
拖动节点     → 自由移动位置
滚轮缩放     → 放大/缩小视图
拖动背景     → 平移整个网络
暂停/恢复    → 控制物理仿真
```

## 📂 项目结构

```
src/
├── components/          # Vue 组件 (4 个核心 + 2 个页面组件)
├── views/              # 页面视图 (2 个主要页面)

├── services/           # API 服务层
├── router/             # 路由配置
└── styles/             # 全局样式

文档: 5 份详细的中文文档
```

## 🐛 常见问题

**Q: 图表加载很慢？**  
A: 这是正常的。系统已自动应用性能优化，查看左侧"性能优化说明"了解详情。

**Q: 如何修改数据？**  
A: 编辑 `src/views/` 中的页面文件，修改 `generateMockData()` 函数或连接真实后端。

**Q: 支持哪些浏览器？**  
A: Chrome、Firefox、Safari、Edge 的最新版本，需要 ES6 支持。

更多常见问题见 **ARCHITECTURE.md**
